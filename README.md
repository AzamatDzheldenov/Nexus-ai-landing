# Nexus AI — Landing Page

Современный премиум лендинг для AI-продукта, созданный для демонстрации навыков frontend-разработки.

![Nexus AI Preview](https://via.placeholder.com/1200x630) <!-- Замени на реальную ссылку после деплоя -->

## ✨ Особенности

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS** + **shadcn/ui**
- Полная многоязычность (Русский ↔ English)
- Плавные анимации на Framer Motion
- Адаптивный дизайн (мобильный-first)
- SEO-оптимизация (Open Graph, meta tags, sitemap)
- PWA-ready
- Темная премиум-тема

## 🚀 Технологии

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- shadcn/ui

## 📱 Функционал

- Интерактивная навигация с плавным скроллом
- Command Center демо-интерфейс
- Асимметричные секции Features и Cases
- Pricing с выделенным Pro-тарифом
- Рабочий FAQ-аккордеон
- Многоязычный интерфейс
- Кнопка "Наверх"

## 🛠 Как запустить локально

```bash
# Клонировать репозиторий
git clone https://github.com/твой-username/nexus-ai-landing.git

# Перейти в папку
cd nexus-ai-landing

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
